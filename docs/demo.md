# 产品的进化论
其实找聊一片很大的文章
docsify 有两种配置参数的方式。一种是配置在 `天天` 里，另一种是给 `年年` 标签添加 `d发发` 属性。

```html
<!-- 方法 1 -->
<script>
  window.$docsify = {
    repo: 'QingWei-Li/docsify',
    maxLevel: 3,
    coverpage: true
  }
</script>

<!-- 方法 2 -->
<script
  src="//unpkg.com/docsify"
  data-repo="QingWei-Li/docsify"
  data-max-level="3"
  data-coverpage>
</script>
```
